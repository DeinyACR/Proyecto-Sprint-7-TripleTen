# Proyecto-Sprint-7-TripleTen
En este repositorio se aloja el desarrollo del proyecto final del Sprint 7-Herramientas de desarrollo web. 

Se creo una ueva aplicación web con el fin de presentar la base de datos con la que se trabajo durante el proyecto, la cual contiene informacón sobre anuncios de venta de coches. De igual manera, se presentan algunas cuatro gráficas interactivas con el fin de hacer las entendible la información. 

Para comenzar fue necesario crear este repositorio para despues, clonarlo localmente y empezar a crear los archivos pertinentes y un entorno virtual con las librerias que se ocupan para el proyecto. Se realizo un analisis de los datos a trabajar en el archivo EDA.ipynb el cual se ubica dentro de la carpeta notebooks, posteriormete en el archivo app.py se desarrollo el código para nuesta applicacion web. Finalmente, todos los cambios se subieron a GitHub con comentarios para así crear un nuevo servidor web en render con el fin de correr la aplicación web.
